# EOL 

A binary to fetch the API of endoflife.date

## Setup 

```
go install github.com/shoxxdj/eol@latest
```

## Usage

```
EOL.Date : a binary to fetch the API of endoflife.date
	-c: Cycle
	-f: Format : table,inline(default)
	-p: Product
```
